# @gemtest/ui

Shared component library for visual consistency. React + Tailwind CSS.

## Rules

- **Files:** PascalCase (`Button.tsx`) in `src/components/`.
- **Styles:** Tailwind classes only. No CSS Modules.
- **Components:** Atomic and reusable. No business logic or direct API calls.
- **Config:** Apps must scan this package in their `tailwind.config.js`.
